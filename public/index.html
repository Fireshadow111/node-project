API Routes

Method Endpoint Description
GET https://node-project-1-3d99.onrender.com/products Route for all the products in the API to be shown on the products page
GET https://node-project-1-3d99.onrender.com/users Route for all the users in the MySQL table
POST https://node-project-1-3d99.onrender.com/products Route for adding a product to the API
POST https://node-project-1-3d99.onrender.com/users Route for adding users to the API
PATCH https://node-project-1-3d99.onrender.com/users/(user_ID) Route for editing the user in the MySQL table by its id
PATCH https://node-project-1-3d99.onrender.com/products/(prod_ID) Route for editing the product from the API by its id
DELETE https://node-project-1-3d99.onrender.com/users/(user_ID) Route for deleting the user in the MySQL table by its id
DELETE https://node-project-1-3d99.onrender.com/products/(prod_ID) Route for deleting the product from the API by its id
